NOS FORMATIONS




Formation de sensibilisation au phisihing: 350€/4H

Les campagnes de phishing visent à obtenir de la part des utilisateurs des informations sensibles liées à l’entreprise (adresse mail, mot de passe, coordonnées bancaires, etc..).

  • Objectif:
    • Présenter les techniques utilisées par les attaquants et leurs impacts sur l’entreprise.
    • Lister les bonnes pratiques à suivre.
  • Public concerné:
    • Tout personnel de l’entreprise (aucune connaissance technique n’est requise).


Formation sur les bonnes pratiques Active Directory: 2600€/25H

  • Objectif:
    • Présenter les ressources de l'Active Directory
    • Illustrer différentes attaques issues d’une mauvaise configuration
    • Lister les recommandations et bonnes pratiques de durcissement de l’active directory
  • Public concerné:
    • Directeur du système d’information
    • Administrateurs système


Analyse des opération de sécurité Microsoft: 2400€

Cette formation permet aux participants d'acquérir les compétences et connaissances nécessaires pour atténuer les cybermenaces à l'aide de Microsoft Azure Sentinel, Azure Defender et Microsoft 365 Defender. Ces derniers configureront et utiliserot Azure Sentinel et utiliser Kusto Query Language (KQL) pour effectuer la détection, l'analyse et la création de rapports.
